## 定价逻辑：倍率是什么意思？

DuckCoding 用的是"倍率计费"，和直接按 token 算钱有点不同，但其实理解起来不难：

> 你的账户有多少美元额度，API 调用时按官方 token 价格乘以倍率来扣费。
> 倍率 1.5× 就是官方价的 1.5 倍；倍率 0.6× 就是打了六折。

**高峰时段**（工作日 12:00–20:00）vs **优惠时段**（工作日 20:00–次日 12:00，以及节假日全天），倍率是不一样的。

对于白天上班、晚上写个人项目的开发者来说，**把重型任务排到晚上跑**，能省不少。
